# Almohandes Google Play Release Package

This folder contains the Android App Bundle, Play Store graphics, screenshots, listing copy, release notes, and signing fingerprint material for publishing المهندس on Google Play.

## Main Files

- `build/almohandes-release-v2.0.0-4.aab`
- `graphics/app_icon_512x512.png`
- `graphics/feature_graphic_1024x500_google_play.png`
- `graphics/feature_graphic_1024x512_requested.png`
- `screenshots/*.png`
- `metadata/google_play_listing.md`
- `metadata/fast_review_checklist.md`
- `signing/upload_key_fingerprints.txt`

The private keystore credentials are stored under `signing/`; do not publish that folder publicly.
